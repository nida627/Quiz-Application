# Quiz-Application

A simple console-based Quiz Application developed using Python. This project allows users to take a multiple-choice quiz, calculate scores automatically, and view quiz results. The application demonstrates the use of Python fundamentals such as dictionaries, loops, functions, and conditional statements.

## Features

* Take a multiple-choice quiz
* Automatic score calculation
* View individual student result
* View all student results
* Menu-driven console application
* User-friendly interface

## Technologies Used

* Python 3.x
* Visual Studio Code
* Windows Operating System

## Concepts Used

* Variables and Data Types
* Dictionaries
* Loops
* Functions
* Conditional Statements
* User Input and Output
* Menu-Driven Programming
